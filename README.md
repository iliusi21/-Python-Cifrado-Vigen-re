# Cifrado Vigenère en Python

Este programa permite cifrar y descifrar mensajes utilizando el método de cifrado Vigenère con el abecedario en español (incluye la letra ñ).

## Requisitos

- Tener instalado Python 3

## Cómo ejecutar el programa

1. Descarga el repositorio o clónalo.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta el siguiente comando:

```bash
pythonvigenere.py
